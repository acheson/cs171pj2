Instructions

All ares of this visualization are designed to be explored by the user. Consisting of three main views, the viewer may choose which data to explore with the radio buttons at the top of the page. Clicking the animate button will automatically transition between these three views. The List and Scatter Plot allow the viewer to filter the titles and data shown on the map. List titles can be selected by clicking, multiple by click-drag. More granular selections made by command or control clicks. The scatter plot supports brushing and seletion of indiviudal titles. Both the map and bar chart will highlight individual elements and present more detailed site information. The map itself is zoomable with the mouse wheel.

Description

Illicit online movie streaming websites are designed to obfuscate who is responsible for sharing content while also providing viewers with free access to movies and TV shows. Streaming sites serve as indexes for third-party hosted movies and TV shows by linking to content rather than hosting it themselves.

To document and visualize this decentralized sharing structure, we scraped a highly-ranked indexing site, 1channel.ch, for links to hosts for the top 250 movies on IMDB. With this information, we captured which hosts were hosting which movies, and how many viewers had viewed each link. We compiled all this activity into a visualization with three views. The first view shows the total number of times all the films in our dataset have been viewed on 1channel.ch. The second view shows the geographic location of all the different hosts that host the content, and how many views each site has received. Finally, the last view shows the number views that came from each country, based on the Alexa percentage of visitors by country data for each host.

More details about our project and process can be found in our process book.
